# How to Add Screenshot

## Step 1: Take Screenshot

1. Run your app: `streamlit run app.py`
2. Take a screenshot:
   - **Windows**: `Windows + Shift + S`
   - **Mac**: `Cmd + Shift + 4`
   - **Linux**: `PrtSc`
3. Save as: `screenshot.png`
